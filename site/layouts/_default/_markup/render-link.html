{{- /* render-link.html — Hugo markdown render hook. Overrides how every markdown [text](url) link renders across the site. External links (source citations to techcommunity.microsoft.com, msrc.microsoft.com, etc.) open in a new tab so readers don't get carried off modernworkweekly.com. Internal/relative links (e.g. "/exec/", "/deadlines/", anchor links within the same post) render as plain links with no target override, since they're supposed to navigate in-place. */ -}} {{- $u := urls.Parse .Destination -}} {{- if $u.Host -}} {{ .Text | safeHTML }} {{- else -}} {{ .Text | safeHTML }} {{- end -}}
