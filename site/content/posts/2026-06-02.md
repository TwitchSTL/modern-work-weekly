## Top 5 This Week

1. **Active npm supply chain attacks targeting dev and CI/CD environments** — Two separate campaigns (dependency confusion via 33 malicious packages, and the Mini Shai-Hulud typosquatting campaign) are actively harvesting credentials from GitHub, AWS, Kubernetes, Vault, npm, and 1Password. If your build pipelines run npm install without lockfile enforcement or package integrity validation, you are exposed right now.

2. **Microsoft Agent 365 is generally available** — The control plane for governing, observing, and securing AI agents at scale has shipped. If you're running Copilot Studio agents in production, this is your primary tool for agent sprawl management and security posture. Get the updated Administering and Governing Agents whitepaper v3.2 alongside it.

3. **Computer-using agents in Copilot Studio are GA** — Agents can now drive UI-based workflows in legacy web apps and vendor portals without APIs. This is a significant capability unlock, but it also means agents can take real actions in systems outside your normal governance perimeter. Review your Copilot Studio agent policies before users start pointing these at sensitive line-of-business applications.

4. **Three new AI model tiers land in M365 Copilot simultaneously** — Claude Opus 4.8, GPT-5.5 Instant, and GPT-5.5 Thinking are all now available or rolling out. GPT-5.5 Thinking (Reasoning) is in Copilot Studio early release environments. Model availability affects data residency and compliance posture — validate your DLP and sensitivity label configurations cover Copilot outputs across these new models.

5. **The Gentlemen ransomware: aggressive self-propagating Go encryptor** — Storm-2697 affiliate ransomware combines per-file ephemeral key encryption with simultaneous lateral movement techniques across the entire network. Read the full analysis and validate your lateral movement detection rules and network segmentation posture.
