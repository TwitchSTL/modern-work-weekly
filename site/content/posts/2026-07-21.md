## Top 5 This Week

1. **Exchange 2016/2019 ESU ends October 2026, no extension coming.** The Exchange team has confirmed there will be no Period 3. If you have on-premises Exchange 2016 or 2019, your migration or upgrade timeline now has a hard floor.

2. **ACR Stealer ClickFix campaigns are actively targeting enterprise environments.** From late April through mid-June 2026, Defender Experts observed sustained campaigns stealing browser credentials, auth tokens, and sensitive documents. User awareness and endpoint controls need to be current.

3. **AsyncAPI npm supply chain compromise used CI/CD pipelines to deliver malware.** Threat actors weaponized trusted workflows for import-time payload delivery. Any organization running Node.js-based tooling or pipelines pulling npm packages should audit dependencies now.

4. **Microsoft Entra Internet Access and Private Access extend Zero Trust to AI agents.** The preview update covers AI-agent-originated traffic, not just user and device sessions. If you are deploying AI workflows that make outbound calls, this is the access control layer to evaluate.

5. **Eight Linux kernel CVEs published across GPU, KVM, netfilter, and audio subsystems.** All are informational publications this week. Scope is Linux-side, but relevant to any Defender for Endpoint Linux deployments or Azure-hosted Linux workloads.

## Security & Compliance

- **[CVE-2026-60082: DBI versions before 1.651 for Perl - statement handle consistency](https://msrc.microsoft.com/update-guide/vulnerability/CVE-2026-60082)** [GA] - Information published by MSRC this week. Relevant if your environment includes Perl-based tooling or automation that uses DBI; check your DBI version and update to 1.651 or later.

- **[CVE-2026-63801: tipc - slab-use-after-free in tipc_aead_decrypt_done](https://msrc.microsoft.com/update-guide/vulnerability/CVE-2026-63801)** [GA] - Information published. TIPC is a Linux kernel networking protocol; relevant to Linux workloads running Defender for Endpoint or hosted on Azure. Monitor for kernel patch availability from your distribution.

- **[CVE-2026-63882: drm/amdkfd - NULL pointer bug in svm_range_set_attr](https://msrc.microsoft.com/update-guide/vulnerability/CVE-2026-63882)** [GA] - Information published. AMD GPU kernel driver issue; affects Linux hosts running AMD hardware. Relevant to Azure VMs or physical endpoints using AMD GPUs under Linux.

- **[CVE-2026-63879: drm/amdgpu - fix amdgpu_hmm_range_get_pages](https://msrc.microsoft.com/update-guide/vulnerability/CVE-2026-63879)** [GA] - Information published. A second AMD GPU driver issue in the same subsystem as CVE-2026-63882; evaluate both together when assessing patch applicability on AMD Linux hosts.

- **[CVE-2026-64077: netfilter ebtables - two-stage removal scheme](https://msrc.microsoft.com/update-guide/vulnerability/CVE-2026-64077)** [GA] - Information published. Netfilter is foundational to Linux network filtering and containerized workload security. Track vendor patches for Linux distributions used in your environment.

- **[CVE-2026-63940: KVM SEV - Ignore Port I/O requests of length '0'](https://msrc.microsoft.com/update-guide/vulnerability/CVE-2026-63940)** [GA] - Information published. Relevant to KVM-based virtualization with AMD SEV (Secure Encrypted Virtualization). If you operate or connect to KVM-hosted confidential compute workloads, track this against your hypervisor vendor's patch cadence.

- **[CVE-2026-64097: drm/amd/display - Validate GPIO pin LUT table size before iterating](https://msrc.microsoft.com/update-guide/vulnerability/CVE-2026-64097)** [GA] - Information published. AMD display driver bounds validation fix in the Linux kernel. Low direct enterprise risk unless AMD GPU Linux endpoints are in scope, but worth logging for your patch tracking.

- **[CVE-2026-64133: ALSA asihpi - Fix potential OOB array access at reading cache](https://msrc.microsoft.com/update-guide/vulnerability/CVE-2026-64133)** [GA] - Information published. Out-of-bounds read in the ALSA HPI audio driver for Linux. Niche hardware scope, but any Linux endpoint running this driver should be queued for a kernel update.

- **[Secure AI, web, and private apps with Zero Trust](https://techcommunity.microsoft.com/t5/microsoft-entra-blog/secure-ai-web-and-private-apps-with-zero-trust/ba-p/4516387)** [Preview] - Microsoft Entra Internet Access and Private Access now extend Zero Trust policy enforcement to AI agent-originated traffic, not just user and device sessions. If your organization is deploying Copilot agents or other AI workflows that make outbound or internal API calls, evaluate this as your access control layer before those workloads scale. Review the preview enrollment steps and assess how your current Conditional Access posture maps to agent identities.

- **[Reminder: Exchange 2016 and 2019 ESU Program Ends in October 2026](https://techcommunity.microsoft.com/t5/exchange-team-blog/reminder-exchange-2016-and-2019-esu-program-ends-in-october-2026/ba-p/4539033)** [GA] - The Exchange team has confirmed there will be no Period 3 ESU extension: once October 2026 ends, no further updates ship for Exchange 2016 or 2019, period. If you still have on-premises Exchange, accelerate your migration to Exchange Online or upgrade to Exchange Server 2019 Subscription Edition immediately. This is a hard deadline with zero runway for renegotiation.

- **[Microsoft at Black Hat USA 2026: Defending trust in the age of AI and supply chain attacks](https://www.microsoft.com/en-us/security/blog/2026/07/17/microsoft-at-black-hat-usa-2026-defending-trust-in-the-age-of-ai-and-supply-chain-attacks/)** [GA] - Microsoft Security is presenting supply chain research and hands-on sessions at Black Hat USA 2026. Worth tracking the session content for threat intelligence on AI-era supply chain attack patterns that will influence defender tooling roadmaps.

- **[ACR Stealer: Two observed intrusion chains amid increased threat activity](https://www.microsoft.com/en-us/security/blog/2026/07/16/acr-stealer-two-observed-intrusion-chains-amid-increased-threat-activity/)** [GA] - From late April through mid-June 2026, Defender Experts observed sustained ACR Stealer campaigns using ClickFix lures to harvest browser credentials, authentication tokens, and sensitive documents from enterprise environments. Review the two intrusion chains documented in the blog, validate your endpoint detection coverage for ClickFix-style social engineering, and confirm browser credential theft detections are active in your Defender for Endpoint policies.

- **[Unpacking the AsyncAPI npm supply chain compromise and import-time payload delivery](https://www.microsoft.com/en-us/security/blog/2026/07/15/unpacking-asyncapi-npm-supply-chain-compromise-import-time-payload-delivery/)** [GA] - Threat actors compromised AsyncAPI npm packages and abused CI/CD pipelines to deliver malware at import time, bypassing typical runtime detection. If your organization uses Node.js tooling, developer pipelines, or any dependency on AsyncAPI packages, audit your npm dependency trees and lockfiles now. Review CI/CD pipeline permissions and consider enforcing package integrity checks as a standing control.

## Action Required

- **[Exchange 2016/2019 ESU - Hard Deadline: October 2026](https://techcommunity.microsoft.com/t5/exchange-team-blog/reminder-exchange-2016-and-2019-esu-program-ends-in-october-2026/ba-p/4539033)** [GA] - Deadline is end of October 2026, and the Exchange team has explicitly ruled out any further extension. Identify every on-premises Exchange 2016/2019 server in your estate, assign a migration or upgrade owner, and establish a completion milestone before October. Organizations still on Exchange 2016 should treat this as a P1 migration workstream.

- **[ACR Stealer ClickFix campaigns - Active threat, immediate review advised](https://www.microsoft.com/en-us/security/blog/2026/07/16/acr-stealer-two-observed-intrusion-chains-amid-increased-threat-activity/)** [GA] - Active campaigns are successfully exfiltrating browser credentials and tokens from enterprise environments right now. Validate that Defender for Endpoint behavioral detections for ClickFix lures are enabled and alerting, review recent credential-related alerts in Defender XDR, and push a user awareness reminder on social-engineering download prompts.

- **[AsyncAPI npm supply chain compromise - Audit pipelines within 30 days](https://www.microsoft.com/en-us/security/blog/2026/07/15/unpacking-asyncapi-npm-supply-chain-compromise-import-time-payload-delivery/)** [GA] - Any development or DevOps pipeline consuming npm packages is a potential exposure point. Within the next 30 days, audit CI/CD pipeline configurations for overly permissive package resolution, enforce lockfile integrity, and review whether AsyncAPI packages appear in any direct or transitive dependencies across your toolchain.
